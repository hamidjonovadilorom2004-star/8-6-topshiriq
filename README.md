# Polyclinic Blockchain System (6-chi topshiriq)

Bu loyiha poliklinika bemorlari uchun elektron tibbiy kartalarni boshqarish va xavfsiz blokcheyn asosida ma'lumot almashish tizimidir.

## Talab qilingan funksiyalar:
*   **Elektron tibbiy karta**: Bemor ma'lumotlari (tashxis, davolash) blokcheynda saqlanadi.
*   **Access Control**: Bemor o'z ma'lumotlarini qaysi shifokorga ochishini (Grant/Revoke) o'zi belgilaydi.
*   **Audit Log**: Kim, qachon ma'lumot qo'shganini blokcheyn orqali kuzatish mumkin.
*   **Dori tekshiruvi**: Batch ID orqali dori vositalarining haqiqiyligini tekshirish imkoniyati (Medicine Verification).
*   **Premium React DApp**: RainbowKit va Wagmi orqali zamonaviy tushunarli interfeys.

## Steklar:
- **Solidity**: ERC-modellar asosida tibbiy mantiq.
- **Vite/React**: Tezkor va interaktiv interfeys.
- **Emerald Theme**: Sog'liqni saqlashga moslashtirilgan yashil-emerald dizayn.

## Qanday ishlatish mumkin:
1. `cd 6_chi_topshiriq`
2. `npm install`
3. `npm run dev`

Loyiha butun dunyo uchun ochiq bo'lishi maqsadida GitHub'ga yuklab, shartlar to'liq bajarilgan.
